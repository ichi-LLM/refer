🚀 JAMA要件管理ツール

JAMAとExcelを連携させて、要件管理を効率化するツール

📌このツールでできること

📥 要件の取得

JAMAプロジェクトから要件を階層構造で取得し、見やすいExcel形式で出力します。

✏️ 簡単な編集

使い慣れたExcelで要件を編集。新規追加、更新、削除が直感的に行えます。

📤 一括更新

編集した内容をJAMAに一括反映。ドライラン機能で安全に更新できます。

⚡クイックスタート

1

必要なパッケージをインストール

pip install requests openpyxl
2

設定ファイル（config.json）を作成

API IDとAPI Secretを設定します

3

テンプレートを作成（練習用）

python main.py template -o practice.xlsx
4

要件を取得

python main.py fetch -o requirements.xlsx

💻主なコマンド

📄 テンプレート作成（NEW!）

💡 JAMAに接続せずに、練習用のExcelテンプレートを作成できます
python main.py template -o template.xlsx

📥 要件の取得

目的 コマンド
プロジェクト全体 python main.py fetch -o all.xlsx
特定の場所から（sequence） python main.py fetch -o part.xlsx -s 6.1.5
特定の場所から（名前） python main.py fetch -o part.xlsx -n "公共駐車場"
階層制限あり python main.py fetch -o shallow.xlsx -d 3

📤 要件の更新

python main.py update -i requirements.xlsx --dry-run

↑ まずドライランで確認（実際の更新はしません）

python main.py update -i requirements.xlsx

↑ 問題なければ実行

📊Excelファイルの編集方法

操作の種類

新規 JAMA_IDを空欄にして、新しい要件を追加
更新 既存の要件の内容を変更
削除 操作列に「削除」と入力

Description編集（5行形式）

📝 SYSPのDescriptionを詳細な5行形式のテーブルで編集できます
  1. 「Description更新」列を「する」に変更
  2. 「新Description参照」列のリンクをクリック
  3. Description_editシートで5行形式のテーブルを編集

❓よくある質問

Q: ドライランって何ですか？

A: 実際にJAMAを更新せずに、どんな変更が行われるかを確認する機能です。本番前のリハーサルのようなものです。

Q: 大量のデータを更新しても大丈夫？

A: 必ずドライランで確認してから実行してください。また、重要なデータはバックアップを取ることをお勧めします。

Q: エラーが出たらどうすれば？

A: jama_tool.logファイルに詳細なエラー情報が記録されています。それを確認してください。

⚠️注意事項

🔑 API Secretは一度しか表示されません。必ずメモを取ってください。
💾 大量の要件を更新する前には、必ずJAMAのバックアップを取ってください。
